## Pytorch LSTM_GCN 图卷积神经网络_火车票识别项目

本课程介绍用图卷积神经网络(GCN)，实现一个「火车票文字信息识别提取」的项目。实验证明，LSTM+GCN网络结构，在身份证、护照、驾驶证、发票、购物小票等证件和票据识别上，都有非常好的效果。本项目是作者工作中的真实项目，只需要结合自己的工作场景略作修改，就是一个可以在面试中大大加分的商业项目。
